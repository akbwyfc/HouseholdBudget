# Household Budget

A lightweight household budgeting application written in Delphi 12 Community Edition.

## Features

- SQLite database
- FireDAC
- Income & Expense tracking
- Monthly Budgets
- Reports
- CSV Export
- Backup & Restore

## Requirements

- Delphi 12 Community Edition
- SQLite
- Windows 10/11
